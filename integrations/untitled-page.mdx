## Step 1: Go to Event Type Settings

Navigate to your Cal.com event type settings.

![Event Type Settings]()

## Step 2: Open Booking Questions

Click on "Booking Questions" in the event type settings.

![Booking Questions]()

## Step 3: Add a Question

Click "Add a question" to create a new booking field.

![Add Question]()

## Step 4: Configure the Field

Set the following values:

| Setting      | Value        |
| ------------ | ------------ |
| `Input Type` | Short Text   |
| `Identifier` | `utm_fp`     |
| `Hidden`     | Enabled      |
| `Required`   | No           |

![Field Configuration]("/add-custom-utm-param-hidden-question.png")

## Step 5: Save

Save the event type to apply your changes.

That's it! The Thred script will automatically handle the rest.

## Important

<Warning>
Make sure you have the Thred tracking script installed on your website before configuring Cal.com. Without the script, the integration will not work.
</Warning>
