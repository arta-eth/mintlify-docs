Thred helps GTM teams surface buyer intent from AI search conversations across ChatGPT, Gemini, Claude, and Perplexity.

## Docs Structure

This documentation has four core sections:

- [Website Script](/script-tracking/ai-search-signals): install the tracking script on your site
- [Conversations API](/getting-started/get-enriched-chat-from-customer): retrieve conversations and insights
- [MCP Server](/mcp/overview): connect Thred tools in Cursor, Claude Desktop, and other MCP clients
- [Integrations](/integrations/calcom): configure downstream integrations

## 1) Website Script

Install the script to start capturing AI search signals:

```html
<script src="https://cdn.thred.dev/thred-track.js?browserKey=YOUR_BROWSER_KEY"></script>
```

Start here: [Track AI Search Signals](/script-tracking/ai-search-signals)

## 2) Conversations API

Use the API to retrieve conversation transcripts and insights.

Base URL:

```bash
https://api.thred.dev/v1
```

Authentication:

```bash
Authorization: Bearer YOUR_API_KEY
```

| Use case                      | Endpoint                                             |
| ---------------------------- | ---------------------------------------------------- |
| Fetch by email               | `GET /customers?email=<email>`                       |
| Fetch by customer ID         | `GET /customers/:customerId`                         |
| Fetch by company             | `GET /customers/by-company?name=<company_name>`      |
| Fetch recent conversations   | `GET /customers/recent`                              |

## 3) MCP Server

Connect the hosted MCP endpoint in your client:

```text
https://mcp.thred.dev/v1?apiKey=YOUR_THRED_API_KEY
```
