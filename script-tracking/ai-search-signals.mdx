## Install Script

Add this script tag to your site, ideally before the closing `</body>` tag:

```html
<script src="https://cdn.thred.dev/thred-track.js?browserKey=YOUR_BROWSER_KEY"></script>
```

## What It Enables

After installation, Thred captures AI search buyer activity and helps connect downstream form submissions to those AI-originated buyers.

## Verify Installation

1. Deploy the script on your site.
2. Open your site in a browser.
3. Confirm the request to `cdn.thred.dev/thred-track.js` is successful in the Network tab.
4. Validate new activity appears in Thred.
